# YT_LSTM
YT_Title_Predict

We developed a Bi-LSTM Model to predict the relationship between the Title and the views of YouTube videos.


## Usage
### Model Training
#Get data from Youtube  
`$ python Youtube_Crawler.py`  
`$ python model_F.py`
